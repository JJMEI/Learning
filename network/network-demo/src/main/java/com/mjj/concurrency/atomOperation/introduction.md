   如何实现原子操作？ 
        使用锁一定可以实现原子操作 synchronized 基于阻塞的锁机制 独占式的
        使用锁带来的问题
            1. 被阻塞的线程优先级很高怎么办？
            2. 拿到锁的线程一直不释放锁怎么办？
            3. 大量的竞争，消耗CPU，同时带来死锁或者活锁的问题
            
            其次🔐是一种比较重的解决办法
   
   
   CAS的原理 compare and swap
   CAS指令 CPU级别的指令保证这是一个原子操作 循环这个指令知道成功位置
   三个运算符：
        1. 一个内初地址V
        2. 一个期望的值A
        3. 一个新值B
        
   基本思路：如果地址V上的值和期望的值A相等，就给地址V赋上新值B
            循环里（自旋）里不断的进行CAS操作
            
            
   CAS的问题：
        1. A-->B-->A的问题 解决方案  A1->B2->A3  值没变但是版本号发生变更  
        2. 开销问题 陷入死循环
        3. 只能保证一个共享变量的操作
