【政策解读-公务员晋升】

​​今天我们说一下跟公务员管理密切相关的《职务职级并行规定》。

学生读者、体制内读者和体制外读者都可以读一读。我们尽量用浅显的语言解释的更清楚一些。

 

公务员体制改革是2013年《深化改革若干重大问题决定》中明确要求改革的60大任务中的一项，原话是：深化公务员分类改革，推行公务员职务与职级并行、职级与待遇挂钩制度，加快建立专业技术类、行政执法类公务员和聘任人员管理制度。

 

（一）历史

 

其实大家不要把公务员看的很神秘，也就是社会职业中的一种。涉及公务员管理的制度也并不是一开始就有的，而是陆续才出台的。

 

1.初始。1982 年、1983年，中央和地方国家机关先后进行了机构改革，并按照干部“四化”的方针，调整了各级领导班子，建立了老干部的离休、退休制度，开始逐渐废除实际上长期存在的领导职务终身制，这是公务员管理系统化的起点。

 

2.推动。干部终身制废除了，但是干部录用、考核、交流、培训这些事情怎么办？于是，1984年国务院提出要制定《国家机关工作人员法》，但是“国家机关工作人员”究竟指哪些人？没法定论，所以1986年起草的时候改名为《国家行政机关工作人员条例》，1993年根据情况，更名为《国家公务员暂行条例》。而从1993年开始，全国正式实施公务员制度，从这一年开始，公务员考录制度正式建立了，也就是俗话说的“凡进必考”。

 

3.延伸。1983年-1993年期间，国家还制定了很多零散的条例来规范公务员的行为，比如，1989年的《国家行政机关工作人员贪污贿赂行政处分暂行规定实施细则》、1991年的《国家行政机关工作人员回避暂行规定》等等，这些条例都没有上升到法律范畴。2006年，为了与当时国家法治化建设相匹配，正式出台了《公务员法》，把职位分类、录用、考核、任免、职务升降、奖惩、培训、交流、回避、申述控告等全部纳入这部法律。这也算是正式有了一个说法。

 

4.深化。2013年，中央深化改革决定中，就明确提出要把公务员改革纳入改革的重要内容。在2014年以后，按照中央部署，在激励公务员方面，分别推出了新的《干部选拔任用条例》、《公务员法》和《关于县以下机关建立公务员职务与职级并行制度的意见》等文件，在管理公务员方面，就是一系列从严管党治党的文件，这里就不赘述了，因为成果大家都看到了，非常显著。

 

上述脉络比较简单清晰的把历史描述了一遍。

 

（二）问题

 

公务员管理存在哪些问题？

 

具体来说，分为正反两个方面。

 

1.正向方面。就是对公务员激励手段不够。

 

在历次的改革中，都出现一个问题，就是“开前门”的办法较少。什么叫做“开前门”，也就是没有文件明确提出基层干部晋升怎么办？工作付出回报怎么办？养老怎么办？

 

大家可能不是很能理解。

 

实际上，在之前的法律规定中，公务员职务分为领导职务和非领导职务，对于职级的管理非常弱化。

 

非领导职务与领导职务相比，无论晋升空间还是各类待遇，都处于弱势，要不然大家也不会说“当官不带长，放屁都不响”。

 

由于职级功能的弱化，职务晋升成了公务员价值追求的最大激励和职业发展的唯一阶梯。党报的调查显示，81%的被调查者认为职务晋升对公务员的激励最有效，其中领导职务的晋升比非领导职务的晋升更具有吸引力。

 

这里再插入一句，大家对于职务和职级也不要认为很玄乎，其实互联网公司最喜欢干的就是这个。

 

举个例子：阿里巴巴的体系中，P7就是职级，对应的职务呢？管理类可能叫做经理，技术类可能叫做专家。这里的管理类就是公务员的领导职务，技术类就是非领导职务。而无论经理还是专家，作为P7级的职级，就对应的是50-70万的年收入（记忆不一定准确，大概是这个意思）。

 

这样容易理解吧，公务员也是一样。

 

换句话说，职级对应职务，职务对应收入。

 

根子还在收入上。

 

坦白的说，我国的公务员是宝塔形结构，越往上职务越少，尤其是基层公务员，职务晋升难的问题更为突出。

 

目前，我国公务员中有超过60%在县以下基层工作，有超过76.7%的属于乡科级及以下人员。并且，在基层单位里，99%的公务员属于副处级以下，90%的公务员属于副科以下。

 

所以，处级干部对于基层公务员来说就是一辈子的指望。

 

而1993年当时的《国家公务员暂行条例》中，提到了实施的“职务级别工资制”，但问题在于，基于职务级别的基本工资在公务员职级工资制度中的占比过小（35%左右），同时基本工资相对于单位的津贴、补贴也显得过小，所以公务员还是感觉不到级别作用，只能依靠职务。

 

由于职务对于公务员来讲，既决定着晋升，也决定着待遇，还决定着个人身份、地位和自我价值感、认同感，因而在激烈的竞争压力下，公务员队伍管理暴露出不少问题，最大的问题就是：腐败，比如，跑官卖官，比如，“59岁现象”等等。

 

而不涉及到腐败的人员，就会出现另外一种情况，因升迁无望，丧失工作动力，人浮于事，倚老卖老，这在基层特别明显，在县乡一级大部分公务员的年龄一超过40岁或45岁，因为上升空间受阻，产生“应付工作”的情绪。

 

2.反向方面。就是从严管党治党，从严管理干部。也就是“堵后门”

 

这个就不用赘述了。

 

从纪委监委口，出来的报导都很多，既包括查处处理干部，又包括对下的督导监督等等方面。

 

（三）办法

 

怎么在从严管党治党的同时，加大对干部的激励呢？

 

两个办法。

 

第一，正向激励

 

1.修订《公务员法》

 

《公务员法》从法律层面对过去存在的问题进行了定性，并固化下来。

 

2018年1月1日实施的新《公务员法》在四个方面有了巨大改变。

 

一是明确了非领导职务改革。这次明确规定，综合管理类公务员职级层次分为四等十二级：一级巡视员、二级巡视员、一级调研员、二级调研员、三级调研员、四级调研员、一级主任科员、二级主任科员、三级主任科员、四级主任科员、一级科员、二级科员。

 

而以前的规定是非领导职务：巡视员、副巡视员、调研员、副调研员、主任科员、副主任科员、科员、办事员。

 

换句话说，对非领导职务进行了规范，而正科副科，变成了一级二级三级四级主任科员，晋升更容易，也更快。

 

二是工作满三十年可申请提前退休。《公务员法》第九十三条明确规定：工作满三十年的，可以申请提前退休。符合国家规定的可以提前退休的其他情形也可申请提前退休，这种情况一般是指特殊岗位的公务员，由于其岗位特殊的工作条件，为保护其健康，国家允许其提前退休。

 

这也解决了公务员队伍中人浮于事，对稳定队伍军心起到反作用的部分人群处理的问题。

 

三是基层公务员工资有隐性提升。由于晋升速度加快，因此基层公务享受待遇的速度也会加快，对于稳定基层公务员有利。

 

四是加班明确提出有补助。明确规定：公务员在法定工作日之外加班的，应当给予相应的补休，不能补休的按照国家规定给予补助。这对“5+2”、“白+黑”的基层来说也是非常大的利好，不至于每次加班都白干。

 

2.职务职级并行

 

职务职级工作重点解决是非领导职务问题，也就是重点解决干部积极性的问题。它是《公务员法》的具体执行规定。

 

这项工作实际上是分了两步走。

 

第一步是2014年，先出台了《关于县以下机关建立公务员职务与职级并行制度的意见》，这是主要面向县以下机关，重点是稳定基层公务员。

 

这一次调整，基本上把公务员调整的方向定调了：将公务员经济待遇的分配主要由职务决定转向主要由职级决定，强化职级对经济待遇的决定功能。解决“千军万马过独木桥”，挤在干部提拔这一条路上的问题。

 

这个意见执行了4年，情况还算比较好。所以2018年就在全范围内推行职务职级并行，不仅针对县级以下，还针对中央机关、省市两级的机关人员（这些人的积极性也很重要）。

 

这一次的制度最大的特点是两个：

 

一是晋升机会多了。总体而言，央、省、市、县四级的晋升机会普遍增加。

 

中央层面：

 

中央机关一级、二级巡视员不超过机关综合管理类职位数量的12%。其中，

正部级单位一级巡视员不超过一级、二级巡视员总数的40%

副部级单位一级巡视员不超过一级、二级巡视员总数的20%

一级至四级调研员不超过机关综合管理类职位数量的65%

 

改革前：中央机关层面，巡视员和副巡视员职数，不得超过该机关厅局级领导职务职数的三分之一，其中巡视员不得超过巡视员和副巡视员职数的40%；调研员和副调研员职数，不得超过县处级领导职务职数的75%。

 

省级机关层面：

 

省、自治区、直辖市机关一级、二级巡视员不超过机关综合管理类职位数量的5%，其中，

一级巡视员不超过一级、二级巡视员总数的30%，一级至四级调研员不超过机关综合管理类职位数量的45%

 

改革前：省直机关巡视员和副巡视员职数，不得超过厅局级领导职务职数的三分之一，其中巡视员不得超过巡视员和副巡视员职数的30%；调研员和副调研员职数，不得超过县处级领导职务职数的50%。

 

市级机关层面：

 

市（地、州、盟）、直辖市的区领导班子一级巡视员不超过领导班子职数的15%。市（地、州、盟）、直辖市的区机关二级巡视员不超过机关综合管理类职位数量的1%。

一级至四级调研员不超过机关综合管理类职位数量的20%，其中一级、二级调研员不超过一级至四级调研员总数的40%，一级调研员不超过一级、二级调研员总数的50%；

一级至四级主任科员不超过机关综合管理类职位数量的60%，其中一级、二级主任科员不超过一级至四级主任科员总数的50%。

 

改革前：地级市机关层面，调研员和副调研员职数，不得超过县处级领导职务职数的三分之一，其中调研员不得超过调研员和副调员职数的30%;主任科员和副主任科员职数，不得超过乡科级领导职务职数的50%。

 

县级层面：

 

县（市、区、旗）领导班子二级巡视员不超过领导班子职数的10%，一级、二级调研员不超过领导班子职数的20%。

县（市、区、旗）、乡镇机关二级调研员不超过机关综合管理类职位数量的2%；

三级、四级调研员不超过机关综合管理类职位数量的10%，其中三级调研员不超过三级、四级调研员总数的40%；

一级至四级主任科员不超过机关综合管理类职位数量的60%，其中一级、二级主任科员不超过一级至四级主任科员总数的50%。

 

二是有利于基层。这几年，“逃离乡镇、逃离基层”的心态在快速抬头，这一次职务职级就体现了稳定基层的导向。

 

从条件看：

1.晋升一级巡视员，应当任厅局级副职或者二级巡视员4年以上；

2.晋升二级巡视员，应当任一级调研员4年以上；

3.晋升一级调研员，应当任县处级正职或者二级调研员3年以上；

4.晋升二级调研员，应当任三级调研员2年以上；

5.晋升三级调研员，应当任县处级副职或者四级调研员2年以上；

6.晋升四级调研员，应当任一级主任科员2年以上；

7.晋升一级主任科员，应当任乡科级正职或者二级主任科员2年以上；

8.晋升二级主任科员，应当任三级主任科员2年以上；

9.晋升三级主任科员，应当任乡科级副职或者四级主任科员2年以上；

10.晋升四级主任科员，应当任一级科员2年以上；

11.晋升一级科员，应当任二级科员2年以上。

 

从晋升年限来看，很明显有利于基层干部，不利于省直机关、部委机关年轻人，比如：

 

乡镇机关公务员最高可以晋升到正处级的二级调研员，职业空间往上提升了两个档次。而且，本科毕业进入公务员定二级科员，只需要两年即可提拔副科级的四级主任科员。相比以前的三年，减少了一年。

 

换句话说，读三年制的研究生，不如本科毕业直接进入公务员。节约一年，还多了两年的工作经验。

 

再比如：省直、部委机关，从副科待遇晋升为副处待遇，以前最低六年，现在是最低八年。

 

但是这次制度也有几个潜在的问题：

 

一是非领导收入比领导收入高，导致不容易平衡。比如说，很多地方一级调研员的工资比处长还高，但处长是实职，承担的压力巨大又累很多，谁还愿意当处长？

 

二是事业编、编外人员怎么办？目前反映比较强烈的是编外人员和教师、医生等事业编制人员，特别是各地教师队伍，呼吁非常强烈。

 

第二，反向约束

 

也就是继续加大治党管党的力度，这个也不赘述了，属于纪委监委的范畴，新闻报道非常多。

 

总体而言，职务职级并行在全国范围内推广，已经算是迈出了第一步，总比没有要强很多。

 

后续的修订或者是推广到其他事业单位等，需要看具体执行效果。

 

今天就说到这。
